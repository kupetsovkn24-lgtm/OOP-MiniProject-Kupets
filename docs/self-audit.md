## Обрана траєкторія IW29

Добудовую три пов'язані теми:

1. Делегат `Func<Job, bool>` як гнучка умова відбору завершених робіт.
2. LINQ-аналітика, що на основі відбору формує звіт дорогих ремонтів.
3. Тести й демонстрація цього звіту через наявний console-сценарій.

Цей вибір посилює реальну потребу адміністратора СТО: швидко побачити дорогі виконані ремонти та суму виручки по них. Окремий великий модуль або новий патерн для цього не потрібні.
